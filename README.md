# EJP2026-GenAI4teaching
Supplemental Materials for EJP2026-GenAI4teaching


This document provides a concise overview of the manuscript titled "Diagnosing physical self-consistency in multi-stage dynamics via GenAI-assisted interactive visualization" and its accompanying supplementary materials. All resources are openly accessible for non-commercial academic and educational use under appropriate attribution.

Materials Overview

1. GenAI prompt template: A standardized HTML code prompt to generate interactive visualization evolution with real-time energy tracking.

2. Interactive html webpage: One .html file is provided as a complete single-file interactive webpage for visualizing the energy transformation process in the inclined-plane, spring, and collision system.

3. SVG files required for code execution: Eight .svg files are provided as the visual assets required by the interactive html webpage. These SVG files correspond to the staged diagrams of the AB and the AC collision systems, respectively.

4. Staged diagrams of the AB and AC collision systems: 
(1)AB_node1.svg, AB_node2.svg, AB_node3.svg, and AB_node4.svg are corresponding to the  pre-collision instant, the post-collision instant, the max compression of the string, and the separation instant for the block A and the block B, respectively;
(2) AC_node1.svg, AC_node2.svg, AC_node3.svg, and AC_node4.svg are corresponding to the  pre-collision instant, the post-collision instant, the max compression of the string, and the separation instant for the block A and the block C, respectively.
